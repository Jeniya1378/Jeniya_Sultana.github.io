Research and Projects

  • Wearable Sensing and Machine Learning for Construction Safety
    Description: Ongoing PhD research focused on understanding safety, fatigue, and well-being in physically demanding construction tasks using wearable sensing technologies (e.g., EEG, EMG, and related biosignals). The work integrates experimental design, data analysis, and machine learning methods under real-world constraints.
  • GAN-Based Data Augmentation for Biosignal Analysis
    Description: Research exploring generative adversarial networks (GANs) to augment limited and noisy biosignal datasets, with the goal of improving robustness and generalization of machine learning models in safety-critical, human-centered applications.
  • Evaluation of Passive Exoskeletons Using Multimodal Sensing
    Description: Study of the effects of passive exoskeletons on human performance and safety during construction-related tasks, combining wearable sensing, experimental protocols, and data-driven analysis to assess physical and cognitive impacts.
  • Dynamic Context Detection from Spoken Text and Voice Commands in Home Automation Systems
    Description: Identified contextual intent from voice commands and conversational inputs to enable autonomous control in smart environments using SBERT, attention mechanisms, clustering, and zero-shot learning.
  • Context Recognition from Voice Commands in Smart Home Environments
    Description: Developed a framework for extracting contextual meaning from short spoken texts using transfer learning and SBERT-based representations with attention, emphasizing efficiency and low computational cost.
  • Metamorphic Testing for Verification of Neural Network Models
    Description: Designed metamorphic relations to evaluate robustness and reliability of neural network models used for context recognition tasks.
  • Students’ Attention Monitoring Using Facial Landmark Points
    Description: Applied MediaPipe FaceMesh and transfer learning (Xception) to classify attention-related states using the DAiSEE dataset.
  • Research Paper Classification Using Feature Selection and TF-IDF
    Description: Implemented a neural network-based classifier using TF-IDF and feature selection techniques to categorize research papers based on titles and abstracts.
  • Text Analysis Tool for Thematic and Sentiment Analysis
    Description: Developed an end-to-end thematic and sentiment analysis tool designed for non-technical users.
  • Tracking System for Humans with Carried Baggage
    Description: Designed a hybrid computer vision tracking system using Mean Shift, Particle Filter, and Sliding Window techniques to balance speed and accuracy.
  • School Database Management System
    Description: Designed and implemented a database-driven school management system using PHP, MySQL, HTML, CSS, and JavaScript.
  • Library Management System
    Description: Developed a web-based library management system using HTML, CSS, WordPress, PHP, and MySQL.
  • Employee Management System
    Description: Built an employee management platform using PHP and MySQL to support administrative workflows.
  • Online Bus Ticket Reservation System
    Description: Created a web-based booking platform for online bus ticket reservations.
  • Online Food Ordering System
    Description: Developed a food ordering system integrating Google APIs for location-based services.
  • App Recommendation System
    Description: Implemented a recommendation system using usage data and AI-based techniques.
  • Day/Night Transformation
    Description: Computer graphics project implementing day-to-night scene transformation using C++.
  • Simple Arithmetic Calculator
    Description: Developed a calculator supporting arithmetic expressions and matrix operations.
  • Crude Oil Price Prediction
    Description: Built an LSTM-based time-series prediction model using historical crude oil price data (1987–2019).
