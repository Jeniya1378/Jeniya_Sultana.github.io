Research and Projects

Research Projects

  • Dynamic Context Detection from Spoken Text and Voice Command in Home Automation System
    Description: Identifying contexts from voice commands and conversations with smart assistants to enable autonomous control and adaptability in smart home environments using SBERT, attention, clustering, and zero-shot approaches.
  • Context Recognition from Voice Commands in Smart Home
    Description: Framework to extract contexts from short-spoken texts using transfer learning, SBERT encoding, and attention with low computational cost.
  • Application of Metamorphic Testing for Verification of Neural Network Model in Home Automation System
    Description: Designed metamorphic relations to validate robustness of a neural network context recognition model.
  • Students Attention Monitoring System using Facial Landmark Points
    Description: Used MediaPipe FaceMesh and transfer learning (Xception) to classify attention-related states on DAiSEE dataset.
  • Research Paper Classification: A Neural Network Approach with Feature Selection and TF-IDF
    Description: TF-IDF + feature selection + neural network classifier for categorizing research papers using metadata (titles/abstracts).
  • Text Analysis Tool: Thematic and Sentiment Analysis
    Description: End-to-end tool for thematic and sentiment analysis for non-technical users.
  • Tracking System for Human with Carried Baggage (Mean Shift + Particle Filter + Sliding Window)
    Description: Hybrid computer vision tracking system optimizing speed and accuracy.
