Publications

  1. DyCoDe: Dynamic Context Detection from Voice Commands and Conversations in Smart Homes
    Authors: Jeniya Sultana, Razib Iqbal
    In Proceedings of the 2025 IEEE 22nd Consumer Communications & Networking Conference (CCNC).
    Description: This paper presents a novel dynamic context detection framework, DyCoDe, designed for smart homes. It leverages conversational data to identify and respond to voice commands using SBERT, attention layers, and a zero-shot classification technique, allowing the system to adapt to new IoT environments.
    Link
  2. Metamorphic Testing for Investigation of Context Recognition from Smart Home Voice Commands
    Authors: Fahima Hasan Athina, Jeniya Sultana, Debaleen Das Spandan, Razib Iqbal
    In Proceedings of the 2024 7th Conference on Cloud and Internet of Things (CIoT).
    Description: This paper introduces a metamorphic testing approach to evaluate a context recognition model for smart home voice commands. The framework proposes 10 metamorphic relations to verify and validate the model's accuracy.
    Link
  3. A Framework for Context Recognition from Voice Commands and Conversations with Smart Assistants
    Authors: Jeniya Sultana, Razib Iqbal
    2024 IEEE 21st Consumer Communications & Networking Conference (CCNC), IEEE, 2024, pp. 218-221.
    Description: This paper proposes a novel framework for extracting contexts from short-spoken texts using transfer learning, a fully connected neural network aided with SBERT encoding, and an attention mechanism.
    Link
  4. Misinformation Detection in Cyber Smart Cities
    Authors: Anupom Mondol, Jeniya Sultana, Mohiuddin Ahmed, ANM Bazlur Rashid
    Cybersecurity for Smart Cities: Practices and Challenges, Springer International Publishing, 2023, pp. 105-120.
    Description: This book chapter discusses the pervasive nature of misinformation, its impact on smart cities, and various detection techniques.
    Link
  5. Masters Thesis: Towards Dynamic Context Detection From Voice Commands and Conversations With Smart Assistants in Smart Homes
    Author: Jeniya Sultana
    Description: Developed a two-step approach: (1) context recognition model using transformer + attention, and (2) a dynamic context detection framework that groups new conversational data so the system can learn new contexts.
